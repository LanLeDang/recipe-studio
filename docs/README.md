# Recipe Studio

Recipe Studio is a web-based platform designed to organize and showcase recipes in a visually appealing and user-friendly way. Whether you're a seasoned chef or a beginner in the kitchen, Recipe Studio provides inspiration and guidance for creating delicious meals. The site features recipes organized into categories like appetizers, main dishes, and desserts, along with contact information for the creators.

## Setup

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/username/recipe-studio.git
   ```
2. Then, navigate to the project repo:
   cd recipe-studio

### FEATURES

- **Responsive Design**: The site is fully responsive and adapts to different screen sizes.
- **Recipe Categories**: Recipes are organized into categories like appetizers, main dishes, and desserts.
- **Interactive Recipe Cards**: Each recipe card includes an image, description, and a link to the full recipe.
- **Hero Section**: A visually appealing hero section introduces the site.
- **Contact Page**: Includes contact information for the creators with clickable links to email, LinkedIn, and GitHub profiles.

## Purpose

The purpose of Recipe Studio is to provide a clean and organized platform for showcasing recipes. It was created as part of a web development project to demonstrate skills in HTML, CSS, and responsive design. The project also aims to inspire users to explore new recipes and improve their cooking skills.

## Implementation

- **HTML**: Used to structure the content of the site, including sections for recipes, contact information, and navigation.
- **CSS**: Styled the site with a focus on responsive design, using Flexbox and CSS Grid for layout.
- **CSS Grid**: Implemented for the recipe cards to create a dynamic, responsive grid layout.
- **Flexbox**: Used for the overall page layout, including the header, footer, and main content.
- **Hero Section**: Designed with a background image and overlay for a professional look.
- **Contact Page**: Styled with card-like designs for contact options, including hover effects for interactivity.
